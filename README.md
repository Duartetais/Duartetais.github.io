## 📝 Sobre o Projeto
Este repositório contém o código-fonte do meu portfólio profissional. O objetivo principal é centralizar meus projetos, habilidades técnicas e experiências em um único lugar, facilitando o contato com recrutadores e outros desenvolvedores.

---

## 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, utilizei tecnologias fundamentais da web para garantir performance e uma base sólida:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, layout responsivo e animações.
*   **JavaScript (ES6+):** Manipulação de DOM e interatividade.

---

## ✨ Funcionalidades
*   **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktop.
*   **Seção de Projetos:** Listagem dinâmica dos meus principais trabalhos no GitHub.
*   **Formulário de Contato:** Integração direta para mensagens profissionais.
*   **Dark/Light Mode (Opcional):** Alternância de temas para melhor experiência visual.

---

## 🚀 Outras Tecnologias e Ferramentas
Além das linguagens principais, este projeto e meu fluxo de trabalho envolvem:

*   **Git & GitHub:** Controle de versão e hospedagem.
*   **Linux (Fedora):** Ambiente de desenvolvimento principal.
*   **GitHub Actions:** Automação de deploy para o GitHub Pages.
*   **Google Fonts:** Tipografia personalizada para melhor leitura.
